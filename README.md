# Alya Proxy

CORS Proxy untuk ElevenLabs dan Groq API.

## Cara pakai

Ganti URL API di web kamu:

**ElevenLabs TTS:**
```
https://YOUR-PROXY.vercel.app/api/proxy?url=https://api.elevenlabs.io/v1/text-to-speech/VOICE_ID
```

**Groq Whisper:**
```
https://YOUR-PROXY.vercel.app/api/proxy?url=https://api.groq.com/openai/v1/audio/transcriptions
```

**Groq Chat:**
```
https://YOUR-PROXY.vercel.app/api/proxy?url=https://api.groq.com/openai/v1/chat/completions
```
